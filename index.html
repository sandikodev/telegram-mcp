📦 Quick Setup
# 1. Clone & install
git clone https://github.com/sandikodev/telegram-mcp.git
cd telegram-mcp && bun install

# 2. Authenticate (one-time)
bun run auth

# 3. Add to Kiro CLI (~/.kiro/settings/mcp.json)
{
  "mcpServers": {
    "telegram": {
      "command": "bun",
      "args": ["run", "/path/to/telegram-mcp/src/index.ts"]
    }
  }
}
5 Powerful Tools
Everything your AI agent needs to work with Telegram
telegram_get_messages

Read message history from any chat or forum topic — including private groups

telegram_send_message

Send formatted messages to any chat or forum topic with Markdown support

telegram_send_document

Upload and send any file — PDFs, CSVs, images — to any chat or topic

telegram_get_dialogs

List all your chats, groups, and channels with unread counts

telegram_get_topics

List all forum topics in a supergroup — perfect for organized workspaces
